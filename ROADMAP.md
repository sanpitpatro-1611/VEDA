# VEDA Programming Language Roadmap

# Current Keywords

| Keyword   | Purpose                 |
| --------- | ----------------------- |
| say       | Display output          |
| remember  | Store variables         |
| ask       | User input              |
| change    | Update variables        |
| calculate | Perform calculations    |
| if        | Conditional execution   |
| otherwise | Alternative execution   |
| repeat    | Repeat commands         |
| action    | Create reusable actions |
| do        | Execute actions         |

# Long-Term Vision

VEDA aims to become a beginner-friendly programming language that combines readability with practical software development.

The long-term goal is to build:

* VEDA Language
* VEDA Compiler
* VEDA Runtime
* VEDA Package Manager
* VEDA Studio (Official IDE)
* Open Source Community
* Cross-platform Support
* AI-powered Development Tools
